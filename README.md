# Task Management System

## Commands

### For Frontend
```sh
npm run dev

Database

create the database and put the db name to .env
Run Migrations: 
  php artisan migrate

Seed the Database: 
  php artisan db:seed

Start the Server:
  php artisan serve

API Endpoints

For Admin
Admin Dashboard: GET http://localhost:8000/admin
Manage Users: GET http://localhost:8000/admin/users/
Add User: POST http://localhost:8000/admin/users/create
Edit User: PUT http://localhost:8000/admin/users/{user}/edit

For User
Register: POST http://localhost:8000/register
User Login: POST http://localhost:8000/login
